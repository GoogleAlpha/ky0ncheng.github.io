Chrome33+ Adkill的失效解決方法

這個PPAPI版flash插件是經過Google加工過的 所以版本號略奇葩 返回到chrome32自帶的ppapi flash plug-in就可以了

Windows 32位用戶

C:Program FilesGoogleChromeApplication33.0.1750.117PepperFlash

Windows 64位用戶

C:Program Files (x86)GoogleChromeApplication33.0.1750.117PepperFlash

下載地址 http://pan.baidu.com/s/1jG4vWSQ

 

Mac用戶 自行提取 /Applications/Google Chrome.app/Contents/Versions/32.xxxxx(你的chrome32的程序文件夾)/Google Chrome Framework.framework/Internet Plug-Ins/PepperFlash/PepperFlashPlayer.plugin

Linux用戶 下載 http://pan.baidu.com/s/1bnGf7h9 覆蓋至/opt/google/chrome/pepperflash下
