Linux是如何建立的







平臺之爭：Linux vs Windows vs 蘋果系
同宗之爭：Linux vs BSD
桌面之爭：Ubuntu N種桌面 vs MacOS桌面 vs Windows 桌面
版權之爭：自由軟件基金會 GPL證書類 vs Android聯盟 Apache證書/BSD證書類 vs Windows/蘋果閉源世界
開放性之爭：Linux 強制開源 vs Windows、蘋果多數閉源
版本兼容性之爭：Linux 多變 Kernel、API 接口 vs 穩定API接口的Windows、Mac、iOS
驅動程序(版權+開放性+版本兼容性)之爭：Linux 驅動程序老、少、弱 vs Windows 驅動程序新、多、強 vs 蘋果獨家使用
嵌入式跨平臺之爭：Linux 廣泛CPU、設備支持 vs Windows有限支持 vs 蘋果獨家使用
安全性之爭：Linux 極少病毒 vs Windows多病毒 vs Android不少病毒 vs 蘋果極少病毒
費用之爭：Linux 免費 vs Windows 高額費用
市場之爭：Linux 多數用在服務器 vs Windows 多數用在桌面

HTML 5 <video> 標籤



## 定義和用法

<video> 標籤定義視頻，比如電影片段或其他視頻流。




## 實例

一段簡單的 HTML5 視頻：

<video src=”movie.ogg” controls=”controls”>您的瀏覽器不支持 video 標籤。</video>

親自試一試




## HTML 4.01 與 HTML 5 之間的差異

<video> 標籤是 HTML 5 的新標籤。




## 提示和註釋

提示：可以在開始標籤和結束標籤之間放置文本內容，這樣老的瀏覽器就可以顯示出不支持該標籤的信息。




## 屬性












































屬性值描述
autoplayautoplay如果出現該屬性，則視頻在就緒後馬上播放。
controlscontrols如果出現該屬性，則向用戶顯示控件，比如播放按鈕。
heightpixels設置視頻播放器的高度。
looploop如果出現該屬性，則當媒介文件完成播放後再次開始播放。
preloadpreload如果出現該屬性，則視頻在頁面加載時進行加載，並預備播放。如果使用 “autoplay”，則忽略該屬性。
srcurl要播放的視頻的 URL。
widthpixels設置視頻播放器的寬度。




## 全局屬性

<video> 標籤支持 HTML 5 中的全局屬性。




## 事件屬性

<video> 標籤支持 HTML 5 中的事件屬性。
