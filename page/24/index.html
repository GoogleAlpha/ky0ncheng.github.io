Chrome 33+ 自建 擴展 實現 custom.css

實現步驟：
1、新建一個文件夾，比如customCSS，包含custom.css和manifest.json，custom.css就是新版被閹割的那個文件
2、manifest.json的內容如下

{
   "content_scripts": [ {
      "css": ["custom.css"],
      "matches": [ "http://*/*", "https://*/*" ]
   } ],
   "description": "custom.css",
   "name": "custom.css",
   "version": "1.0",
   "manifest_version": 2
}
description是擴展簡介，name是擴展名字，version是版本號，都可以自己改 3、打開chrome的擴展管理，選擇打包擴展程序，擴展程序根目錄選擇customCSS文件夾，私有密匙文件不用管，然後選擇打包擴展程序，生成customCSS.crx和customCSS.pem密匙 4、將customCSS.crx拖入擴展管理安裝即可，任務管理器裏面無進程 擴展模板下載 [http://pan.baidu.com/s/1c0zO3G8](http://pan.baidu.com/s/1c0zO3G8) 以後還要修改可以去userdata裏找到擴展文件夾 修改css文件即可 分享我的custom.css一個

Chrome33+ Adkill的失效解決方法

這個PPAPI版flash插件是經過Google加工過的 所以版本號略奇葩 返回到chrome32自帶的ppapi flash plug-in就可以了

Windows 32位用戶

C:Program FilesGoogleChromeApplication33.0.1750.117PepperFlash

Windows 64位用戶

C:Program Files (x86)GoogleChromeApplication33.0.1750.117PepperFlash

下載地址 http://pan.baidu.com/s/1jG4vWSQ

 

Mac用戶 自行提取 /Applications/Google Chrome.app/Contents/Versions/32.xxxxx(你的chrome32的程序文件夾)/Google Chrome Framework.framework/Internet Plug-Ins/PepperFlash/PepperFlashPlayer.plugin

Linux用戶 下載 http://pan.baidu.com/s/1bnGf7h9 覆蓋至/opt/google/chrome/pepperflash下
