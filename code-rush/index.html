Code Rush 奔騰的代碼

推薦使用 OpenGG.BAE 屏蔽視頻廣告

這是一部講述Netscape和Mozilla的故事的紀錄片。Netscape是一家偉大的公司，它發明了img標籤、cookie、ssl安全協議，當然還有如今HTML5時代的明星語言Javascript！但是由於衆所周知的原因，它失敗了。Netscape最終將瀏覽器代碼開源，這個新項目就是Mozilla！攝製團隊橫跨了期間重要的幾個時間點，整整跟蹤了程序員們一年的時間，最終制成這部紀錄片。

這是一部關於Netscape公司的紀錄片。

如果你不知道這家偉大的公司，那麼我告訴你，它是瀏覽器和其他許許多多東西的發明者，比如顯示圖片的img標籤、http協議中的cookie、互聯網加密協議SSL、以及javascript語言。上個世紀90年代中期，它的Netscape Communicator瀏覽器，曾經佔據了90%的市場份額。

但是很不幸，它的對手是Microsoft。微軟投入超過1億美元，僱傭1000多個程序員，開發出了IE瀏覽器。更致命的是，微軟將IE免費發行，並且捆綁在操作系統中。

所以，到了1998年的年初，所有人都看出來了，Netscape遇到大麻煩了。

《Code Rush》（奔騰的代碼）這部紀錄片，就是講述Netscape公司在1998年的故事，攝製組整整跟拍了一年。

在其中，你可以看到兩件歷史性大事的第一手畫面。

1998年3月31日，Netscape宣佈將Netscape Communicator瀏覽器開源，這個新項目就叫做Mozilla。

1998年11月24日，由於股價不斷創出新低，Netscape公司在絕望之中把自己以42億美元的價格賣給了AOL。一家曾經如此輝煌的公司，就這樣死了。後來的歷史證明，AOL以42億美元買來的只是一具無用的屍體。我不知道，歷史上有沒有比這更不合算的交易了。但是，這筆交易的好處是，AOL因此成爲Mozilla項目的最大讚助者，Netscape已死，Mozilla活了下來。

Amazon上一位觀衆的留言標題就是”一家偉大公司之死”。

如果你對軟件業感興趣，希望瞭解程序員的生活，以及Mozilla項目的起源和創造它的人們，那麼千萬不要錯過這部紀錄片。

我保證這是一部過目難忘的作品。

著名科技博主阮一峯曾經強烈推薦此片，他說：“如果你對軟件業感興趣，希望瞭解程序員的生活，以及Mozilla項目的起源和創造它的人們，那麼千萬不要錯過這部紀錄片。我保證這是一部過目難忘的作品。”
不關你是正在哪家公司就職，亦或是自己創業，也不論是成功或是失敗，都推薦你看一下本片。

片中有一句話時刻警醒着我們：“你把你自己奉獻給一項事業，而你卻不知道這項工作是否會馬上成爲現實生活中能見到的東西；所以這也是一種很奇怪的諷刺，就是說那些被稱爲是製造未來的人，卻看不到他們自己的未來。”
謹以此片，送給我們的一位好朋友，也送給所有的IT行業從業的朋友！
